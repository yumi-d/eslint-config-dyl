# eslint-config-dyl
eslint规则封装

npm 安装

`
npm install --save eslint-config-dyl
`

项目中新增.eslintrc.js文件，并配置:

`
module.exports = {
  extends: ['dyl'],
};
`

vscode中安装eslint插件，并且开启(点击右下角eslint)，开启不生效的话重启vscode试试
